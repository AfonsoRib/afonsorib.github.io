It's been a while since I had a website but now it seems it's going to stay. After writing my own scripts to generate a static website I gave up when I noticed that I wanted a great amount of features and it would take a while before I got any of that working. I later found out about Hugo and decided to give it a try as it had most of (if not all) the features I was looking for in a static page generator and that brought me to this moment. I forked a theme and started playing around with it and the result is this. I still plan on changing things a little bit but anymore than this is just overkill and I should just leave it for my free time.

A different thing from this blog compared to the other I had is that it is hosted on Github Pages instead of neocities. The change was not due to neocities being bad (it really isn't), I just felt that github just offers a slightly better workflow with git and that's it.

Hopefuly in the coming weeks I get more time to work on the page and maybe some other stuff so that I can start filling up this space with something meaningful.
